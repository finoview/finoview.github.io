# 点评

东证衍生品研究院在每周第一个交易日开市前，通过繁微平台发布周度观点和走势评级判断。

# 策略

东证衍生品研究院持续维护和发布的商品策略，目前主要分为单边和套利策略。

# 流程图

东证衍生品研究院整理发布有关商品领域的产业链流程图和分析框架。

# 行情

展示国内期货交易所上市品种的日度行情，更新存在延迟，具体价量信息以交易所为准。

点击某品种行情的收藏按钮，在个人中心查看收藏的指标

# 报告

东证衍生品研究院发布的分析报告，提供在线预览和下载至本地两种阅览方式。点击【更多】，进入报告搜索分类页，可以按报告种类、发布日期和版块分类筛选报告。

# 新闻

东证衍生品研究院于每个国内期货交易日上午八点半之前，将与外汇、股指以及商品相关的重大新闻整理并推送至繁微平台。
