# 仪表盘目录

图1-3，红色矩形框区域为仪表盘目录，具体包含：

  • 黑色金属
  • 能源化工
  • 有色金属
  • 农产品
  • 贵金属
  • 水文气象
  • 航运物流
  • 建筑建材
  • 外汇
  • 债券
  • 股票

# 关注板块的排序与增删

# 板块简介和标签

图1-5，选择债券板块，点击国债栏目，右侧会展示国债相关的模板，模板上方有板块简介以及国债栏目下面的细分因素，细分因素包括：

  • 价格
  • 持仓
  • 经济增长
  • 通货膨胀
  • 财政和货币政策
  • 流动性
  • 外部因素
  • 风险溢价

选择其他板块，布局类似。
