# 注册/登录

# 注册

在产品介绍页，点击右上角，进入注册页，填写相关信息，进行注册。

注册成功后，默认为二级权限，仅可浏览部分报告（日报、晨报、周报、月报、数据周报），试用期为七天。

# 登录

访问官网时，如果是未登录状态，重定向至产品介绍页；如果是登录超时状态，重定向至登录页。

首先勾选【记住密码】，再选择自动登录的天数。
