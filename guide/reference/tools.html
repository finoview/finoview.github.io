# 智能助手

# 股指期货

智能助手涉及的股指期货功能共有两个，分别为套期保值和预测分红。其中的股指期货套保功能主要面对初次接触股指期货对冲和有意愿对现有股票持仓进行风险管理的投资人，而预测分红对股指期货影响则主要面向对股指期货已经有了较为深刻的理解，并且希望从股指期货的投资中获得更多超额收益的投资人。

通过在智能助手中设立这两大功能模块。一方面，客户能够从以上功能中更为直接的了解到股指期货套保的实际效果，并且经过在不同合约之间的套保效果比对，挑选出较为合适的股指期货套保标的和具体合约。

另一方面，由现货股票分红所造成的股指期货负面影响和其造成的基差损失都将在“预测分红对股指期货影响”功能模块中做可视化展示。专业投资人也因此对基差变动因素中的一部分有了更为详细的了解，最终对期指套保合约的选择以及期现套利策略的收益形成帮助。

# 套期保值

在套期保值功能页面选择现货标的、期货标的、调整窗口时长和对冲比率计算方式之后，可视化界面将展示对冲效果净值图、累积收益率折线图、波动率折线图、最大回撤折线图、夏普率折线图等图表和其他表格。

为了实现以上图表的输出，需要完成以下两个步骤：

  • 若已经构建过现货组合，则可以直接点击选择相应的组合。若想要对全新组合进行套期保值测算，则需通过新建组合选择预设的个股并配置各股权重。

  • 完成对现货组合的建立以后，需要选择窗口内其他参数。其中期货标的的可选参数为IF00、IH00和IC00，分别对应沪深300股指期货、上证50股指期货和中证500股指期货的当月合约连续，对冲比率计算方式的可选参数为OLS、GARCH、GJRGARCH、TGARCH，通过选择不同的对冲比例计算模型，所得到的对冲比例数值也有一定的差异。根据最终对冲效果的对比，投资人可以得到更加符合其套保效果预期的对冲比例大小。

# 预测分红

在选择股指期货品种之后和股票分红预测方法之后，以折线图的方式显示年内股指期货影响的预测值以及实际值的变动情况。预测分红功能主要包括两种不同的分红预测方法，即预测分红法和分红点数预测法，并以此作为依据预测分红可能给股指期货所带来的负面影响。

此功能展示的内容分为两部分数据，包括近一年的历史分红状况数据和分红对股指期货的实际值和预测值数据。

在股指期货分红预测部分，可供选择的预测品种有IF、IH、IC，预测方法有分红点数预测法和股息率预测法。通过选择不同的品种和预测方法，我们可以得到相应股指期货品种在不同月份下受到分红影响而可能产生的贴水现货指数幅度，从而在计算股指期货对冲成本或多头指数增强中剔除各股分红所带来的影响。

具体操作流程为，首先在股指期货品种下拉框中选择目标期货标的，然后再预测方法下拉框中选择分红预测方法，最后点击确认后就可以得到近三年的股指期货分红影响预测值和实际值。
