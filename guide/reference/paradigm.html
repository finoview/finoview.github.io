# 范式

# 封推范式

东证衍生品研究院制作发布的范式模板。

# 权限

如出现无范式权限的提醒，请联系销售人员查看是否可以分配权限。

# 保存至我的范式

点击当前范式的【保存到我的范式】按钮，填写范式名称、范式分组、封面等信息后，可将后台发布的范式保存至个人中心-我的范式。

# 我的范式

# 进入并新建我的范式

从用户头像或个人中心中点击【我的范式】进入我的范式页面。点击【新增范式】按钮创建范式。

# 创建范式分组：

填写名称、选择分组、上传封面后点击确定。

其中范式名称必填；范式分组默认选中当前第一个分组，用户可通过点击‘新建分组’按钮新建分组；范式简介显示在范式内容区域；上传封面则范式应用该封面图片，未上传封面范式应用繁微默认图片。

# 修改范式分组

用户可通过点击范式的编辑按钮，在编辑范式弹窗中，修改范式分组。

# 批量修改范式分组

用户点击我的范式列表中【批量修改分组】按钮，范式封面右上角出现选择框，用户可选择多个范式，选择要修改至的分组后点击确定，选择的范式移至所选分组下。

# 新增定基图

在范式详情页，点击‘新增定基图’按钮，选择指标（选择的指标不能超过10个）移到已选项，点击确定，新增的定基图显示在该页面。当选择指标超过10个，点击确定，系统给出限制提示不允许保存。

# 设置定基条件

  1. 定基图中点击【设置定基条件】，弹出设置弹窗，输入定基条件，点击确定，定基图根据定基条件生成定基后的图表。定基后的图表最多显示十条图例，图例由每年的基点及指标名称组成。

  2. 设置弹窗中，指标开始时间，时间总长度决定图指标的时间范围周期；定基期数决定每年显示的数据数；定基类型为必选项，定基时间/定基事件决定每年定基的日期，点击确定后，根据设置参数生成定基后的图表。

    累计变化值：每个指标每年从定基日期起，之后的每个有数据的日期为一个定基期数，该值减去该年定基日期的值，为本期的累计变化值。

    累计变化率：每个指标每年从定基日期起，之后的每个有数据的日期为一个定基期数，该值减去该年定基日期的值/该年定基日期的值*100%，为本期的累计变化率。

  3. 用户在设置定基参数弹窗中，定基类型选择事件，点击定基事件下拉框右边的新增按钮，输入事件名称、配置基点，点击确定，即可显示在该用户定基事件下拉框中供选择。

# 多图协同

范式详情中点击【多图协同】按钮，弹出设置弹窗，输入定基条件限制同设置定基条件，点击确定后，该定基条件应用于该范式所有定基图。

# 设定坐标

在范式详情页面，点击已设置定基条件的定基图设置坐标按钮，选择指标，设置单位及上下限，对当前图表中该指标的所有图例根据设置描绘定基图。

# 下载PPT

点击下载PPT按钮，弹出选择PPT模板窗口，可选择繁微水印模板或空白模板，点击确定后，生成PPT，名称为范式名称及下载时间，内容为该范式内所有定基图。
