# 工作台

# 页面与操作

  • 进入工作台

    在繁微（finoview）首页中，点击下方任意数据即可进入数据工作台目录。

  • 展开数据列表

    鼠标移动至左侧目录右上方处，可展开更多数据列表。

  • 工作台区域划分

    点击目录内数据源列表，按层级依次展开更多指标，点击其中任意指标，历史数据将以折线图展现在工作台右侧作图区域，并且右下方有该指标的基本说明。工作台上方是功能区域，可实现指标搜索、指标计算、模版保存、图像导出等功能。

# 处理原始指标

# 重要功能介绍

# 功能：变频

  • 功能介绍

    变频就是改变指标的显示频率，例如EIA商业原油库存，如果只想看每月的库存数据，就可以通过变频功能，在改变频率的同时，还可以进行运算，包括求新频率的合计值、平均值，取新频率的第一条或最后一条记录等。

  • 具体操作

    1. 选择“新频率”下拉框，包括：年度、半年度、季度、月度、周度、日度
    2. 选择变频运算
      • 当频率降低时，例如月度数据变成季度数据，选择“降低频率”下拉框
      • 当频率提高时，例如季度数据变成月度数据，选择“提高频率”下拉框

  • 实例：每月最后一周的EIA商业原油库存

    1. 选择指标

      基础----能源化工----原油----美国原油基本面情况----美国原油及成品油库存下找到指标：EIA商业原油库存，增加指标到工作台中。

    2. 选择算法

      点击工作台上端工具栏中的计算图标，选择其中的变频功能

    3. 选择参数

      弹出“变频”对话窗，先选择需要降频的指标。然后，由于频率由周变为月，降低了频率，选择“降低频率”；再选择“新频率”下拉框为每月下拉框，选择“最后一条”(因为是提取每月最后一周的数据) ；最后点“确定”按钮。

    4. 生成指标序列

# 功能：计算变化

  • 功能介绍：

    计算变化就是计算指标比上年同期或上期的增减及增减幅度。例如EIA商业原油库存，可以通过计算变化功能，运算同比或环比的变动幅度或具体值。

  • 具体操作：

    1. 选择变化类型

      • 百分比变化：变化的幅度
      • 数据变化：变化值

    2. 选择变频运算

      • 同比：比上年同月

      • 环比：比指标上一期值

  • 实例：EIA商业原油库存

    1. 选择指标

      基础----能源化工----原油----美国原油基本面情况----美国原油及成品油库存：EIA商业原油库存

    2. 选择算法

      点击工作台上端工具栏中的计算图标，选择其中的计算变化功能

    3. 选择参数

      弹出“计算变化”对话窗，先选择需要计算变化的指标。然后，计算设置为“百分比变动”还是“数据变化”；再选择“同比”还是“环比”；最后点“确定”按钮。

    4. 生成指标序列

# 功能：指标间运算、指标与常数运算

  • 功能介绍

    1. 指标间运算：两个指标间的运算，包括加减乘除、百分比，两个指标的频率必须一样。
    2. 指标与常数运算：指标与常数的运算，包括加减乘除、乘方。

  • 具体操作

    1. 指标间运算：先选择两个指标，然后选择“指标间运算”，并在对话框中选择运算类型。运算顺序由选择指标的先后顺序决定。
    2. 指标与常数运算：选择一个指标，然后选择“指标与常数运算”，在对话框中输入常数后，再选择运算类型。

  • 实例：Brent原油结算价- WTI原油结算价

    1. 选择指标

      1. 基础----能源化工----原油----国际主要原油标的期货价格及价差----美国原油及成品油----ICE Brent：Brent原油结算价(CO1)
      2. 基础----能源化工----原油----国际主要原油标的期货价格及价差----美国原油及成品油----NYMEX WTI：WTI原油结算价(CL1)

      注意：计算Brent原油结算价- WTI原油结算价，必须先点击Brent原油结算价(CO1)加入工作台，然后再点击WTI原油结算价(CL1)加入工作台。

    2. 选择算法

      点击工作台上端工具栏中的计算图标，选择其中的指标间运算功能

      ​

    3. 选择参数

      弹出“指标间运算”对话窗，先选择需要计算变化的指标，再选择计算设置“减”，最后点“确定”按钮。

    4. 生成指标序列

# 功能：季节性分析

  • 功能介绍

    将指标序列直接生成季节性分析图表。新的各年时间序列为日频，若出现空值的情况可选择保留空值或用前值代替空值。

  • 具体操作

    季节性分析：先选择一个指标，然后选择“季节性分析”，并在计算设置中选择“空值”或“延用前值”。

  • 实例：EIA商业原油库存季节性分析

    1. 选择指标

      在基础----能源化工----原油----美国原油基本面情况----美国原油及成品油库存：EIA商业原油库存

    2. 选择功能

      选择点击工作台上端工具栏中的计算图标，选择其中的季节性分析功能

    3. 选择参数

      弹出“季节性分析”对话窗，先选择需要计算变化的指标，再选择计算设置“延用前值”，最后点“确定”按钮。

# 其他

  • 双轴

    双轴图：不同指标数据按照左轴或右轴的刻度显示

  • 图形细节

    图形类型：选择线形图、柱状图或累积柱状图

  • 开始时间：选择指标的开始时间，可分为1年、5年、10年、MAX、年至今和自定义。

  • 保存和分享：图片保存可将工作台所完成的图表下载至本地保存，分享功能则可在微信扫码后分享给好友或微信群。
