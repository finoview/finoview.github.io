# 使用手册

提示

请在登录状态下点击相关元素链接，游客状态下链接将重定向至繁微官网首页。

# 七大元素

  • 数据：标准时间序列，均已清洗并指标化

  • 地图：合成数据指标，以及航运图层可视化

  • 报告：东证衍生品研究院宏观、商品、金工、以及数据组提供定期、专题、调研报告

  • 策略：模型描述、净值展示

  • 新闻：东证衍生品研究院晨报速递，覆盖国内外宏观以及各商品板块

  • 快讯：由唐宋大数据提供钢铁行业的实时资讯服务

  • 会议：由东证衍生品研究院召开的培训和会议

通过首页搜索框检索相关元素，或者通过对应页签查看更多相关内容。

# 核心研究功能

  • 仪表盘：由东证衍生品研究院专业分析师提炼的模板集合
  • 工作台：数据指标来源多达二十多种，覆盖国内龙头产业资讯数据提供商，并尝试探索另类数据指标整合
  • 研究报告：东证衍生品研究院宏观、商品、金工、以及数据组提供定期、专题、调研报告
  • 航运可视化：提供铁矿石、原油、煤炭、大豆、LNG以及LPG的航运流量、在途查询
  • 智能助手
    • 持仓分析
    • 股指期货套保和分红
    • 楼市监控

# 使用说明

  • 游客模式（未登录状态）下仅可以浏览部分页面，推荐注册登录后使用，如有进一步提升权限的使用需求，请联系相关销售获取信息

  • 繁微提供API数据下载服务，请联系相关销售获取信息

  • 模板数据无法直接下载保存

    从工作台自定义、个人中心保存，以及仪表盘呈现的指标或者模板，底层数据在网页上均无法直接下载，只能通过鼠标悬停相关数据线或者数据柱，查看到具体对应数值；

  • 亿海蓝可视化查询结果可以下载保存
