# 繁微指标目录API接口

# 繁微指标目录调用地址

https://finoview.com.cn/autoApi/CMS_CATALOGUE

# 输入用户key和密码

需要以post方式输入以下用户key和密码

字段 字段类型 字段说明
appkey 用户key
appsecret 用户key的密码

# 输出结果表字段

字段 字段类型 字段说明
指标编码 Varchar(50)
指标名称 Varchar(255)
单位 Varchar(50)
频度 Varchar(50)
数据来源 Varchar(100)
备注 Varchar(4000)

# python样例代码

import requests

url = "https://finoview.com.cn/autoApi/CMS_CATALOGUE"
querystring = {"appkey":"请输入APPKEY","appsecret":"请输入APPSECRET"}

requests.packages.urllib3.disable_warnings()
response = requests.request("POST", url, params=querystring, verify=False)
print(response.text)
